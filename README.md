# QuantumTelescope
Quantum Telescope


## Installation
Open terminal and run "npm i" to install project
```
$ npm i
```

## Vite (Development)
To run project run command "npx vite" in project folder root
```
$ npx vite
```
## Vite SSR Build (Production)
Run command to build project
```
$ npx vite build
```
## Node (Development)
Run command to start server after build
```
$ npm start
```
